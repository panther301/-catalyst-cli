Catalyst Tab Group Demo

Catalyst-tab-group is a fully independent Angular library. catalyst-tab-group organizes all tab-panel data into separate views where only one view can be visible at a time. Each tab's label is shown in the tab header. You can fully customize it using CSS classes.

The active tab may be set using the selectedIndex input or when the user selects one of the tab labels in the header. It accepts numbers only.

Why Catalyst Tab Group?

Catalyst-tab-group supports the following features:

  • Lightweight for your project
  • Easy to use
  • Horizontal and Vertical tab layouts supported
  • Pass different classes for tabs and tab-panels to customize
  • Built with Angular Signals for optimal performance
  • Standalone components - no module imports needed
  • OnPush change detection
  • Full accessibility support (ARIA attributes, keyboard navigation)

Inputs

Name Type Default Description
tabGroupConfig TabGroupConfig {{ '{' }} position: 'horizontal', taClass: '', coClass: '' {{ '}' }} Configuration object for tab layout and styling. Controls orientation and CSS classes.
selectedIndex number 0 Index of the currently selected tab (zero-based). Supports two-way binding with [(selectedIndex)]

Outputs

Name Type Description Example

Usage Notes

Two-way Binding: Both components use Angular's model() signal which automatically creates two-way binding. You can use:

  • [(selectedIndex)]="myIndex" - Two-way binding (recommended)
  • [selectedIndex]="myIndex" (selectedIndexChange)="myIndex=$event" - Explicit binding

Accessibility: The component includes ARIA attributes and keyboard navigation (Enter/Space keys) for accessibility.

Signals: All reactive properties use Angular Signals for optimal performance with OnPush change detection.
